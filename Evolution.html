Web 1.0:

Web 1.0 was the first stages of the web. It was made by online a few developers and a lot more users. This was the first stage and showed what the future holds for information sharing. The web just had static pages and if revered to as “the read only web”. The information came form the server filesystem and didn’t use RDBMS to store and retrieve data form. Adds was also banned on websites. The era of web 1.0 was from 1991 to 2004.


Web 2.0:

The focused shifted form reading to participation and contribution on the web. Web 2.0 had many more developers and user generated content for the web. Browser technologies like JavaScript, AJAX, CSS and HTML5 were used to make these websites. This is when the web became social allowing people to connect with each other by sharing their thoughts ,opinions, experiences and much more. It allowed for dynamic content that is responsive to user input. API were developed and introduction towards web application started. Information was starting to flow between the site owner and the site user. This was the time when the web grew from user that just searched for information to a much wider audience. People form all over the world connecting through blogs, sharing content, tagging. It was a shift that was community focused and improved the web by a huge amount from web 1.0.


Web 3.0:

Web 3.0 the web as we know it today. The web that is a massive database and that this information can help the user in there needs at that moment. This is the point where back-end of the web saw some love after web 2.0 only focused on the front-end. The web 3.0 is also know as the Semantic web that promised to establish the worlds information. It improved on the search engine for the user to make it where it will consider the meaning of the words and supply information. Where in the previous versions of the web it was more keyword driven in the supply of information. AI was also introduced that is able to improve over time and supply more relevant information. The introduction of Ciri is also part of web 3.0. The ability to connect most devices with the internet allowing the web to be used on most devices. It make’s use or 3D graphics like in games. Web 3.0 is also very secure and let’s users send messages securely. Even allowing users to perform online banking at the comfort of their home. Web 3.0 is a massive millstone in the evolution of the human race and just to wonder how will the web look in the next 20 years?
