The World Wide Web (more commonly called WWW or just simply “The Web”) is a way for all computers to receive data/information or send data/information. It allows computers all over the world to communicate with each other which allows users to communicate with each other. There are three components of the web: URL (Uniform Resource Locator), HTML (hypertext Markup Language) and HTTP (Hypertext Transfer Protocol).

The World Wide Web has changed drastically since it was first made which has brought many advancements in technology. With the new opportunities there have also been new challenges to go with them. Information nowadays is being used as a currency; knowledge is power. With this new age of connectivity brings a new age of negativity. Privacy is a massive issue in terms of cloud computing, there are debuts to who owns the information stored on the server. The information stored could potentially be worth millions. With cloud computing it also means that the information is on the web and could be accessed by black hat hackers, which brings in new opportunities of cybersecurity.

Definitions

  • URL
    • URL is a commonly used phrase which is also referred to as a link. A URL is a unique identifier for a web page. It allows users to locate and use resources on the internet. A URL can take a user to any resource on the internet whether it be an image, HTML page, a CSS document, etc.
  • HTML
    • HTML is a programming language that is mostly used for web page creation. This is what determines everything on a web page, from the font of the words to the images the user can see on the web page.
  • HTTP
    • HTTP refers to a method of securing and transporting information between computers (such as a server and client).
